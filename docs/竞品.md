## 直接竞品结论

- Fandom 手册是唯一成熟的专属内容库，也是 Steam 直接关联 manual；覆盖面强但以术语说明为主。
- Steam Discussions 提供最及时需求，但 thread 信息分散、版本变化快。
- GameFAQs 和普通数据库站只有 release / platform 摘要，无法解决玩家操作问题。
- YouTube 有官方系统解释与少量独立实机，但尚无覆盖完整正式版的攻略系列。
- 机会不是复制 39 个 Fandom 词条，而是用 17 篇 answer-first 指南覆盖当前高意图问题，并通过 homepage / guide hub / sidebar 建立清晰路径。

## 版本与来源规则

- Steam Store / API 与当前 developer replies 优先。
- Fandom 是 Steam 关联手册，但页面更新时间和完整度不同；使用时保留 URL、checkedAt 与 confidence。
- prerelease 视频只能作为 UI / mechanic visual reference，正文必须与正式版资料交叉核对。
- `disband units`、`food from maize`、性能与第一补丁内容每天检查；计划中的修复不得写成已经部署。
- 不把 SteamDB prerelease cache、第三方 Steam Deck 判断或 online multiplayer 推断当作官方当前事实。
